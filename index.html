Middle Ohio–Laughery Watershed Map

Description

This interactive map shows the hydrology of the Middle Ohio–Laughery Watershed (HU8 05090203), including major rivers, streams, and waterbodies. Labels for flowlines and waterbodies appear when zoomed in. State and county layers provide geographic context, along with a satellite basemap to enhances spatial understanding.

Map Details

Created in Mapbox Studio, with hydrology data from the USGS National Hydrography Dataset (NHD). Data added to QGIS 3.44.2 where it was simplified and exported as GeoJSON before uploading to Mapbox Studio for interactive web mapping. © USGS
